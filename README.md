# leetlink

## Usage

```bash
$ leetlink 207
https://leetcode.com/problems/course-schedule/
```

Pipe it to `open` to open it on your browser:

```bash
$ leetlink 207 | xargs open
[... opens on default browser ...]
```

## Installation

```
go install github.com/marianogappa/leetlink@latest
```
